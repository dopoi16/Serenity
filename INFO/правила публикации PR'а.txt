1. Процесс оформления PR'а:
Прочтение данного документа - создание форка репозитория (если нету) - внесение в него изменений - создание PR'а - обсуждение PR'а с владельцем кода (мной) - далее две концовки
1.1. Одобрение PR'а:
Если по итогу я вынес решение что код действителен - я одобряю его. После чего в кратчайшие сроки ваш код будет внесен в репозиторий. Каждую субботу (иногда бывают задержки) я весь код с репозитория ставлю на сервер.
Если ваш PR одобрили то он будет добавлен в код и вскоре в игру
1.2. Запрос изменений PR'а:
Если по итогу я запросил изменение кода - то это означает что вам нужно следовать моим инструкциям в комментарии либо внести изменения в ваш код, так как по какой-то причине он мне не понравился.
В случае полного отклонения кода PR закрывается и блокируется. Открыть заново его могут только H.Admins (moonash, fedKotikeD)

2. Шаблон PR'а:
PR должен выглядеть по следующему шаблону:


#Описание PR'а
<!-- все что в таких кавычках - комментарии. Не отображается в PR'е -->
<!-- тут вы описываете ваш PR и зачем он нужен -->

Проверки:
[-] PR полностью завершен <!-- обязательно -->
[-] Файлы проверены <!-- обязательно -->
[-] Я добавил медиа <!-- только когда нужно -->
[-] Все файлы протестированы на локальном сервере <!-- необязательно -->

logs:
- added fun <!-- что-то добавили -->
- delete fun <!-- что-то удалили -->
- edit fun <!-- что-то изменили -->
- fix fun <!-- что-то пофиксили -->


Это был шаблон по которому вам необходимо делать PR. PR'ы сделанные без шаблона не администрацией отклоняются сразу!

3. Условия одобрения PR'а:
3.0. (*) - обязательно для одобрения; () - желательно
3.1. Обязательные проверки выполнены (*)
3.2. PR нужный и его хотят люди ()
3.3. PR сделан в стиле сервера ()
3.4. PR не имеет багов (*)
3.5. PR сделан по шаблону (*)
3.6. PR не содержит более 75 коммитов и не более 1000 файлов изменено ()
3.7. Все коммиты названы правильно (*)
3.8. После создания PR'а вы должны через функцию упоминания пригласить создателя кода (fedKotikeD) (*)




Соблюдайте правила публикации пожалуйста!
